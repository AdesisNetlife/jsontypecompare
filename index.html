Description

This is a simple AngularJS webapp that allows you to compare the types of two JSON documents.

The comparison will look at the properties of the document and compare it's types, ignoring the specific values that they may have.

When objects have other child objects, they will get compared too.

About arrays

When an array is found, it will analyze the types of its values. If the values of one array are of differnt type, the array will be marked as of type 'mixed' and it will be considered different to every other arrary. When the array holds object value, the properties of all objects will be analyzed and combined together to define the type of the array.
